## 书评
2020 年 6 月份靠着每日打卡学习啃完了《 HTTP 权威指南》这本厚部头的书，由此可见坚持的力量有多么强大。

这本书出版于 2002 年，当时的 HTTP 版本还是 1.1 版本，这也是为什么书中部分内容已经过时了原因，但这不影响这本书的经典，仍然值得读。
